## How to Play

1. Clone or download this repository.
2. Open `index.html` in a modern browser (Chrome recommended).
3. Allow webcam access when prompted.
4. Hit "Start Game" and use your hands to keep the pixel block "ball" in the air.
5. Attempt to beat your best time! Check the leaderboard for bragging rights.
6. Clear the leaderboard any time with the "Clear Leaderboard" button.
